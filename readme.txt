INSTALLATION INSTRUCTIONS:

Units from the APHI General Purpose Delphi Library are required for this component.  This library may be downloaded from the following website: 
http://www.naadsm.org/opensource/generalpurpose/.  Download and install this library first, following the provided instructions.

Next, double-click the file TARSortGrid.dpk (included with this component).  This will open the package in Delphi.

Click the Install button to install the run time package.  The ARSortGrid component should appear on a palette called APHIControls.

You may also need to include the directory where you save this project in your Delphi Library path (Tools | Environment Options..., Library tab).


DEMO:
A sample application is located in the \Demo directory.
